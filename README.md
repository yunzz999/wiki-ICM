# 📚 Wiki-ICM

Bienvenido a **Wiki-ICM**, un repositorio colaborativo de material de estudio diseñado para estudiantes de matemáticas, enfocado principalmente en la carrera de **Ingeniería Civil Matemática** de la **Universidad de Concepción (UdeC)**.

## 🎯 Objetivo
La misión de este proyecto es democratizar el acceso a la información, creando una "nube" comunitaria donde los estudiantes puedan encontrar y compartir recursos académicos de calidad.

## 📂 Contenido
Dentro del repositorio encontrarás recursos organizados por niveles y asignaturas:

* **Malla Ingeniería Civil Matemática:** Apuntes, guías, certámenes y libros de todos los ramos obligatorios.
* **Licenciatura en Matemáticas:** Material de ramos complementarios y electivos.
* **Recursos Adicionales:** Bibliografía recomendada y herramientas de software.

## 🤝 ¿Cómo contribuir?
Buscamos que este proyecto se mantenga actualizado gracias a los aportes de la comunidad. Si tienes material que quieras compartir (apuntes propios, pruebas antiguas, pautas, etc.), puedes hacerlo de las siguientes formas:

1.  **Fork & Pull Request:** Haz un fork del proyecto, agrega tus archivos en la carpeta correspondiente y envía un PR.
2.  **Issues:** Si encuentras un error o falta material específico, abre un *Issue* para notificarlo.

¡Todo aporte es bienvenido para ayudar a las futuras generaciones de estudiantes!
